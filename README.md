# VeliTurnstile

Cloudflare Turnstile validator module for [Veli](https://github.com/cart96/veli).

## Installation

the package can be installed by adding `veli_turnstile` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:veli_turnstile, "~> 0.1.0"}
  ]
end
```
